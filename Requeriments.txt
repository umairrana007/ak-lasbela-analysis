- Collects daily draw data automatically from websites (GM, LS1, AK, LS2, LS3)

- Uses Hybrid Meta-Learning & Deep Learning models for maximum accuracy
- Generates multi-number predictions with a success-tracking feedback loop
- Integrates structured external knowledge (Rules, Patterns, Hijri Calendar)
- Self-optimizes its own parameters automatically

🤖 6. Advanced Machine Learning System
6.1 Mandatory Models:
- XGBoost, LightGBM, Random Forest
- Time-series models (Prophet)
6.2 Hybrid Meta-Learning (NEW):
- Use a "Stacking Regressor" where a Meta-Model learns how to weigh predictions from individual models based on their recent accuracy.
6.3 Deep Learning Core (NEW):
- LSTM (Long Short-Term Memory) or Transformer-based models to capture long-term sequence dependencies.
6.4 Poisson & Monte Carlo Simulation (NEW):
- Use Poisson Distribution to calculate the mathematical probability of each number (00-99).
- Run 10,000+ Monte Carlo simulations to find the most probable "paths".

🔁 7. Self-Learning & Optimization

- Adjust model weights dynamically (Dynamic Ensemble).
