🎯 Lottery Analysis & Prediction WebApp

Client: Umair | Data Format: 5 Independent Draws (GM, LS1, AK, LS2, LS3) | Range: 00-99 | Version: 2.0

Objective: Build a secure, scalable web app that ingests daily historical draw data, PDF notes, and user-defined patterns. System must analyze each draw independently + optional cross-draw correlations, then generate next-day suggestions with probability scores.

⚠️ MANDATORY DISCLAIMER: Lottery outcomes are fundamentally random. This application is for statistical analysis, pattern tracking, and entertainment only. It does NOT guarantee wins. All outputs must include responsible gaming notices. Never promote gambling or financial risk.

⚙️ Scoring Formula

Final Score (per number per draw) = 
  (Statistical Probability × 0.50) + 
  (Draw-Specific Pattern Match × 0.30) + 
  (Cross-Draw Correlation × 0.10) + 
  (PDF/LLM Insight × 0.10)

Output: Top 10 suggestions per draw (00-99) + confidence % (0-100%)

Let users test two rule configurations side-by-side. System tracks which setup yields higher historical hit rate over time.

📱 PWA & Offline Support

Installable web app. Works offline for viewing cached predictions & history. Syncs when online.

🌍 Urdu/English Toggle

Full i18n support. UI, reports, and disclaimers switch instantly based on user preference.

📤 Smart Export & Share

Export as PDF/CSV. One-click WhatsApp share with formatted prediction card + disclaimer.

🔒 Audit Trail & Versioning

Every rule change, data upload, and prediction generation is timestamped & logged. Rollback support.
